Relatório de Projeto Arduino - Sensor de Cor e LED RGB

Integrantes: Luiz Felipe Guerra, João Pedro Costa Machado, Julio Cesar Florencio, Filipe André Nunes, Felipe Silva da Silva

Objetivo

O objetivo deste projeto é desenvolver um sistema usando Arduino que detecta a presença de cores (preto e branco) através de um sensor UV e controla um LED RGB. Quando o sensor detecta a cor preta, acende uma luz branca no LED; quando a cor branca é detectada, o LED se apaga.

Ferramentas e Componentes

  • Arduino Uno
  • Sensor de cor (TCS3200 ou similar)
  • LED RGB
  • Resistores (220Ω para o LED RGB)
  • Protoboard e fios de conexão
  • Software Tinkercad para simulação

Montagem

A montagem consiste na conexão do sensor de cor ao Arduino, bem como do LED RGB. As conexões são feitas da seguinte forma:

  • Sensor de cor conectado às portas digitais do Arduino conforme datasheet (usando pinos para S0, S1, S2, S3 e OUT).
  • LED RGB conectado nas portas digitais do Arduino com resistores de 220Ω em cada terminal de cor (R, G, B).

Código

O código Arduino a seguir foi desenvolvido para ler a cor detectada pelo sensor. Ele verifica se a cor é preta ou branca e acende ou apaga o LED RGB com base nessa leitura:

#include <Wire.h>
int S2 = 2;
int S3 = 3;
int outPin = 4;
int redPin = 9;
int greenPin = 10;
int bluePin = 11;

void setup() {
pinMode(S2, OUTPUT);
pinMode(S3, OUTPUT);
pinMode(outPin, INPUT);
pinMode(redPin, OUTPUT);
pinMode(greenPin, OUTPUT);
pinMode(bluePin, OUTPUT);
Serial.begin(9600);
}

void loop() {
digitalWrite(S2, LOW);
digitalWrite(S3, LOW);
int freq = pulseIn(outPin, LOW);
Serial.println(freq);

if (freq < 10) { // valor de exemplo para cor preta
digitalWrite(redPin, HIGH);
digitalWrite(greenPin, HIGH);
digitalWrite(bluePin, HIGH);
} else {
digitalWrite(redPin, LOW);
digitalWrite(greenPin, LOW);
digitalWrite(bluePin, LOW);
}
delay(500);
}

Teste no Tinkercad

O projeto foi montado e testado no ambiente de simulação Tinkercad. Durante o teste, variamos as cores presentes na frente do sensor e observamos a resposta do LED RGB, que acendeu ao detectar preto e apagou ao detectar branco, como esperado.

Resultados

Os testes mostraram que o sistema é capaz de detectar corretamente as cores preto e branco. Quando o sensor detecta a cor preta, o LED RGB acende em branco, enquanto ao detectar a cor branca, o LED apaga. O sistema é eficaz e a detecção foi precisa para as configurações estabelecidas.

Conclusão

Concluímos que o projeto atingiu o objetivo de forma satisfatória. Através do uso do sensor de cor e do LED RGB, o sistema demonstrou ser capaz de identificar as cores e responder com base nelas. A simulação no Tinkercad confirmou o funcionamento esperado, e o projeto pode ser expandido para outras aplicações de detecção de cores em projetos de automação e robótica.
