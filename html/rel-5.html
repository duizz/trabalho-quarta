Relatório: Controle de Cor de LED RGB com Potenciômetros no Arduino

Integrantes: João Pedro Costa, Filipe André Nunes, Julio Cesar Florencio, Luiz Felipe Guerra, Felipe Silva da Silva.

1. Objetivo

O objetivo deste projeto é controlar a cor de um LED RGB usando três potenciômetros conectados a um Arduino, onde cada potenciômetro ajusta uma das cores primárias (vermelho, verde e azul). Com isso, o usuário pode criar diversas combinações de cores ao variar a intensidade das três cores de forma analógica.

2. Ferramentas e Materiais Utilizados

  • Arduino Uno (ou qualquer placa compatível)
  • 1 LED RGB (ou 3 LEDs separados: vermelho, verde e azul)
  • 3 Potenciômetros (de 10kΩ)
  • 3 Resistores (de 220Ω)
  • Protoboard (para montagem dos componentes)
  • Fios Jumpers (para conexões)
  • Computador com Arduino IDE (para escrever e enviar o código ao Arduino)
  • Tinkercad (para simulação e teste virtual do circuito)

3. Montagem do Circuito

A montagem do circuito foi realizada conforme as etapas a seguir:

  • LED RGB: Conectamos o cátodo comum do LED RGB ao terra (GND) do Arduino. Os pinos correspondentes às cores vermelho, verde e azul foram conectados aos pinos digitais PWM 9, 10 e 11 do Arduino, respectivamente, por meio de resistores de 220Ω para limitar a corrente.
  • Potenciômetros: Cada potenciômetro foi conectado da seguinte forma:
    • O terminal central (wiper) foi conectado aos pinos analógicos A0, A1 e A2 do Arduino.
    • Um terminal lateral foi conectado ao 5V, enquanto o outro terminal lateral foi conectado ao GND.

Esquema das Conexões:

Componente Conexão com Arduino
LED RGB (Pino R) Pino 9 (PWM)
LED RGB (Pino G) Pino 10 (PWM)
LED RGB (Pino B) Pino 11 (PWM)
Potenciômetro 1 A0
Potenciômetro 2 A1
Potenciômetro 3 A2
Cátodo do LED RGB GND
Terminal lateral Potenciômetros 5V e GND

4. Código do Arduino

int pinRed = 9;
int pinGreen = 10;
int pinBlue = 11;

int potRed = A0;
int potGreen = A1;
int potBlue = A2;

void setup() {
pinMode(pinRed, OUTPUT);
pinMode(pinGreen, OUTPUT);
pinMode(pinBlue, OUTPUT);

pinMode(potRed, INPUT);
pinMode(potGreen, INPUT);
pinMode(potBlue, INPUT);
}

void loop() {
int valueRed = analogRead(potRed);
int valueGreen = analogRead(potGreen);
int valueBlue = analogRead(potBlue);

valueRed = map(valueRed, 0, 1023, 0, 255);
valueGreen = map(valueGreen, 0, 1023, 0, 255);
valueBlue = map(valueBlue, 0, 1023, 0, 255);

analogWrite(pinRed, valueRed);
analogWrite(pinGreen, valueGreen);
analogWrite(pinBlue, valueBlue);

delay(50);
}

5. Teste no Tinkercad

Após a montagem física do circuito, o mesmo foi replicado no Tinkercad, uma plataforma online para simulação de circuitos eletrônicos. No Tinkercad, o circuito foi testado com sucesso, onde ao girar os potenciômetros, a cor do LED RGB mudava conforme esperado, variando a intensidade de cada cor de forma suave.

O Tinkercad permitiu simular o comportamento do código em tempo real, confirmando a funcionalidade da leitura dos potenciômetros e o controle de PWM do LED RGB.

6. Resultados

O projeto alcançou os resultados esperados:

  • Ao girar os potenciômetros, a intensidade de cada cor (vermelho, verde e azul) do LED RGB foi ajustada conforme o movimento do eixo de cada potenciômetro.
  • A combinação de valores das três cores permitiu criar uma ampla gama de cores diferentes, comprovando o controle individual de cada cor via potenciômetros.
  • Quando todos os potenciômetros estavam em valores baixos, o LED permaneceu apagado.
  • Quando todos estavam no valor máximo, o LED emitia luz branca, resultado da mistura de todas as cores em alta intensidade.
  • Combinações intermediárias resultaram em cores como ciano, magenta, amarelo, entre outras.

7. Conclusão

Este projeto demonstrou de maneira eficiente o controle analógico de um LED RGB usando potenciômetros. A utilização dos potenciômetros para ajustar individualmente as cores primárias possibilitou criar inúmeras combinações de cores, o que pode ser aplicado em projetos de iluminação e efeitos visuais. Além disso, o uso de PWM no Arduino para controle de LEDs RGB mostrou-se uma solução eficaz e simples para modulação de intensidade de luz.

O uso da simulação no Tinkercad auxiliou na validação e na depuração do circuito antes da montagem física, o que garantiu um processo de desenvolvimento mais eficiente.
