Relatório 6: Controle de LED com Botão usando Arduino

Integrantes:Luiz Felipe Guerra, João Pedro Costa Machado, Julio Cesar Florencio, Filipe André Nunes, Felipe Silva da Silva

Objetivo

O objetivo deste projeto é construir um circuito simples usando um Arduino, um botão e um LED. Quando o botão for pressionado, o LED acenderá, e quando o botão for solto, o LED se apagará. Este projeto é uma introdução ao uso de botões e LEDs no controle de entradas e saídas digitais em microcontroladores como o Arduino.

Ferramentas e Materiais Utilizados

  • 1 Arduino Uno
  • 1 Protoboard
  • 1 LED (vermelho)
  • 1 Resistor de 220 ohms
  • 1 Botão push-button
  • Fios jumper
  • Cabo USB para conexão com o Arduino
  • Computador com software Arduino IDE

Montagem do Circuito

O circuito foi montado da seguinte forma:

  • O pino positivo do LED foi conectado ao pino digital 3 do Arduino, enquanto o pino negativo foi conectado ao GND através de um resistor de 220 ohms.
  • O botão foi conectado ao pino digital 7 do Arduino e à alimentação de 5V. O outro terminal do botão foi conectado ao GND, utilizando o recurso de INPUT_PULLUP para garantir a leitura correta do botão.
  • Todos os componentes foram conectados ao Arduino e montados em uma protoboard para facilitar o desenvolvimento.

Código do Arduino

const int button_pin = 7;
const int led_pin = 3;
int buttonState = 0;

void setup(){
pinMode(led_pin, OUTPUT);
pinMode(button_pin, INPUT_PULLUP);
}

void loop(){
buttonState = digitalRead(button_pin);

if(buttonState == LOW) {
digitalWrite(led_pin, HIGH);
} else {
digitalWrite(led_pin, LOW);
}
}

Teste no Tinkercad

Antes da implementação física, o circuito foi simulado no Tinkercad, uma plataforma de simulação online. A simulação demonstrou que o botão controlava corretamente o LED, que acendia quando o botão era pressionado e apagava quando era liberado. O comportamento observado confirmou que o código e o circuito estavam funcionais.

Resultados

O circuito funcionou conforme o esperado. Ao pressionar o botão, o LED acendeu, e ao soltar o botão, ele se apagou. Não houve problemas no comportamento do sistema, e a implementação prática correspondeu à simulação feita no Tinkercad.

Conclusão

Este projeto simples demonstrou o uso básico de entradas digitais com o Arduino, usando um botão para controlar um LED. A integração de componentes como resistores, botões e LEDs, junto ao código do Arduino, permitiu o aprendizado prático sobre o controle de dispositivos eletrônicos. O teste no Tinkercad foi uma ferramenta valiosa para garantir que o circuito funcionasse corretamente antes de sua montagem física.
