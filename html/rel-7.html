Relatório 7: Micro Servo e Sensor Ultrassonico HC-SR04

Integrantes: Luiz Felipe Guerra, João Pedro Costa Machado, Julio Cesar Florencio, Filipe André Nunes, Felipe Silva da Silva

Objetivo

O objetivo deste relatório é descrever a implementação de um sistema de controle utilizando um micro servo motor e o sensor ultrassônico HC-SR04, visando medir distâncias e posicionar o servo motor de acordo com os dados coletados.

Ferramentas

  • Arduino Uno
  • Micro Servo Motor
  • Sensor Ultrassônico HC-SR04
  • Fios de conexão
  • Protoboard
  • IDE Arduino

Montagem

A montagem consiste em conectar o sensor HC-SR04 e o micro servo ao Arduino. O pino Trig do sensor deve ser ligado a um pino digital do Arduino, assim como o Echo. O servo deve ser conectado a outro pino digital. A alimentação deve ser feita com 5V e GND.

#include int trig = 12; int echo = 11; Servo s; int pin = 9; void setup() { Serial.begin(9600); s.attach(pin); pinMode(trig, OUTPUT); pinMode(echo, INPUT); s.write(0); } void loop() { long duration; int distance; digitalWrite(trig, LOW); delayMicroseconds(2); digitalWrite(trig, HIGH); delayMicroseconds(10); digitalWrite(trig, LOW); duration = pulseIn(echo, HIGH); distance = duration * 0.034 / 2; Serial.print("Distância: "); Serial.print(distance); Serial.println(" cm"); if(distance <= 50){ s.write(90); }else{ s.write(180); } }

Testes

Os testes foram realizados variando a distância entre o sensor e os objetos. Foi observado que o servo motor respondia de acordo com as distâncias medidas, movendo-se para a posição correspondente ao valor obtido pelo sensor ultrassônico.

Resultados

Os resultados mostraram uma boa precisão nas medições de distância. O servo motor se posicionou corretamente conforme as variações de distância, demonstrando a funcionalidade do sistema integrado.

Conclusão

A integração do micro servo com o sensor ultrassônico HC-SR04 provou ser eficaz para aplicações de medição de distância e controle de movimento. O projeto demonstrou a importância da programação e montagem de circuitos em projetos de automação.
