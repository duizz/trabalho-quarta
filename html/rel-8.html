Relatório de Projeto Arduino: Controle de Micro Servos com Potenciômetros

Integrantes: Luiz Felipe Guerra, João Pedro Costa Machado, Julio Cesar Florencio, Filipe André Nunes, Felipe Silva da Silva

Objetivo

O objetivo deste projeto é controlar o movimento de dois micro servos utilizando dois potenciômetros, um para cada servo. Os potenciômetros atuarão como controladores manuais para ajustar a posição das hélices dos servos, variando o ângulo de 0 a 180 graus.

Ferramentas e Componentes

  • Placa Arduino Uno
  • 2 micro servos (SG90 ou similar)
  • 2 potenciômetros de 10kΩ
  • Protoboard e cabos jumper
  • Software Arduino IDE
  • Cabo USB para conexão do Arduino

Montagem

A montagem consiste em conectar cada potenciômetro ao Arduino e a um dos micro servos. Abaixo estão as instruções de conexão:

  • Conecte o pino VCC de cada potenciômetro ao 5V do Arduino.
  • Conecte o pino GND dos potenciômetros ao GND do Arduino.
  • Conecte o pino de saída de sinal do primeiro potenciômetro ao pino analógico A0 do Arduino, e o do segundo potenciômetro ao pino A1.
  • Conecte o fio de sinal do primeiro servo ao pino digital 9 do Arduino e o do segundo servo ao pino digital 10.
  • Conecte o VCC e o GND de cada servo ao 5V e GND do Arduino.

Código

O código a seguir utiliza a biblioteca Servo.h para controlar o ângulo de cada servo conforme o valor lido nos potenciômetros. O valor analógico de cada potenciômetro é convertido em um ângulo para o servo correspondente.


#include <Servo.h>

Servo servo1;
Servo servo2;

int potPin1 = A0;
int potPin2 = A1;
int servoPin1 = 9;
int servoPin2 = 10;

void setup() {
    servo1.attach(servoPin1);
    servo2.attach(servoPin2);
}

void loop() {
    int potValue1 = analogRead(potPin1);
    int potValue2 = analogRead(potPin2);

    int angle1 = map(potValue1, 0, 1023, 0, 180);
    int angle2 = map(potValue2, 0, 1023, 0, 180);

    servo1.write(angle1);
    servo2.write(angle2);

    delay(15);
}

Teste

Para realizar o teste, siga os seguintes passos:

  1. Carregue o código no Arduino usando a IDE do Arduino.
  2. Após o upload, mova cada potenciômetro e observe o movimento das hélices dos servos.
  3. Ajuste cada potenciômetro lentamente para verificar a resposta de cada servo de acordo com o ângulo desejado.

Resultados

Durante os testes, cada potenciômetro foi capaz de controlar o ângulo de seu respectivo servo, variando de 0 a 180 graus. A resposta dos servos foi suave e rápida, seguindo a posição do potenciômetro quase que instantaneamente. A precisão do controle foi satisfatória para o propósito do projeto.

Conclusão

O projeto foi bem-sucedido em demonstrar o controle de dois servos independentes por meio de potenciômetros. Esse sistema pode ser aplicado em projetos onde controle preciso e manual de ângulo é necessário, como em robótica ou controles de posição. Melhorias futuras poderiam incluir a calibração do sistema para suavizar ainda mais o movimento ou adicionar múltiplos servos e potenciômetros para projetos mais complexos.
