Relatório: Controle de Cor de LED RGB com Sensor de Proximidade

Integrantes: João Pedro Costa, Filipe André Nunes, Julio Cesar Florencio, Luiz Felipe Guerra, Felipe Silva da Silva.

Objetivo:

Desenvolver um circuito utilizando Arduino, sensor de proximidade (ultrassônico) e três LEDs coloridos (vermelho, verde e azul), onde a cor emitida pelos LEDs muda de acordo com a distância medida pelo sensor.

Ferramentas e Materiais Utilizados:

  • Hardware:
    • 1 Arduino Uno
    • 1 Sensor Ultrassônico HC-SR04
    • 3 LEDs (vermelho, verde e azul)
    • 3 resistores de 220 ohms
    • Protoboard
    • Fios de conexão
  • Software: Arduino IDE, Tinkercad

Montagem do Circuito:

A montagem do circuito inclui a conexão do sensor ultrassônico e das três LEDs em diferentes pinos do Arduino.

Código do Arduino:


        const int trigPin = 9;
        const int echoPin = 10;
        
        const int redPin = 13;
        const int greenPin = 11;
        const int bluePin = 12;
        
        long duration;
        int distance;
        
        void setup() {
        
          pinMode(trigPin, OUTPUT);
          pinMode(echoPin, INPUT);
          
        
          pinMode(redPin, OUTPUT);
          pinMode(greenPin, OUTPUT);
          pinMode(bluePin, OUTPUT);
          
        }
        
        void loop() {
          digitalWrite(trigPin, LOW);
          delayMicroseconds(2);
          digitalWrite(trigPin, HIGH);
          delayMicroseconds(10);
          digitalWrite(trigPin, LOW);
        
          duration = pulseIn(echoPin, HIGH);
          
          distance = duration * 0.034 / 2;
        
          Serial.print("Distancia: ");
          Serial.print(distance);
          Serial.println(" cm");
        
          if (distance > 50) {
            setColor(255, 0, 255);
          } else if (distance < 30) {
            setColor(0, 255, 0);
          } else {
            setColor(255, 255, 0);
          }
        
          delay(100);
        }
        
        void setColor(int redValue, int greenValue, int blueValue) {
          analogWrite(redPin, redValue);
          analogWrite(greenPin, greenValue);
          analogWrite(bluePin, blueValue);
        }

Teste no Tinkercad:

O circuito foi simulado no Tinkercad e funcionou conforme o esperado.

Resultados:

O circuito alterou as cores das LEDs com base na distância conforme programado.

Conclusão:

O projeto demonstrou com sucesso o uso do sensor de proximidade para controle de LEDs.
